# Sticker

一个基于 React + GSAP 的前端展示页面。

## 技术栈

- **React 19** — UI 框架
- **Vite** — 构建工具
- **GSAP** — 动效库

## 起步

```bash
npm install
npm run dev
```

## 目录结构

```
src/
  assets/       SVG、图片等静态资源
  components/   公共组件
  styles/       全局样式
  App.jsx       根组件
  main.jsx      入口
```

## 可用脚本

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动开发服务器 |
| `npm run build` | 构建生产版本 |
| `npm run preview` | 预览构建结果 |
| `npm run lint` | 代码检查 |
